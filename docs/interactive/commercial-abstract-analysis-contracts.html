Aerospace & Avionics — DO-178C

The aerospace sector is the most mature market for formal verification. DO-178C (and its tool qualification companion DO-330) mandate rigorous verification of all airborne software. Abstract interpretation has been used in production since 2003.

Landmark achievement: In November 2003, Astrée analyzed 132,000 lines of Airbus A340 fly-by-wire code in 1h20m with zero false alarms. This was the first industrial proof of absence of all runtime errors in a safety-critical system of that scale.

CompCert (Formally Verified Compiler)

Distributed by AbsInt since 2015. Deployed at Airbus Toulouse for over a decade. Qualified by MTU Friedrichshafen for IEC 60880 (nuclear) and IEC 61508-3 SCL 3. CompCert mathematically proves that compiled machine code preserves the semantics of the source C program — eliminating an entire class of compiler-introduced bugs.

▶ SPARK Pro — AdaCore Formal Proof

AdaCore

SPARK Technology

SPARK is a formally verifiable subset of Ada. SPARK Pro uses automated provers (Alt-Ergo, CVC4, Z3) to prove absence of runtime errors, verify functional contracts, and establish information flow properties. Unlike abstract interpretation, SPARK proves full functional correctness of annotated code.

Government & Defense Contracts

Government agencies — particularly DARPA — are the largest single source of formal methods R&D funding. These contracts are often public record and provide the best visibility into contract scope and pricing.

DARPA Formal Methods Portfolio

Scale of investment: DARPA's HACMS program alone was funded at $80M over 4 years. Combined with PROVERS, HARDEN, COOP, CLARA, SafeDocs, and SSITH, DARPA has invested well over $100M in formal methods programs since 2012. This represents the single largest concentration of formal methods funding worldwide.
▶ HACMS — High-Assurance Cyber Military Systems (2012–2016) $80M
DARPA HACMS Program
Making formal methods practical for military vehicle and UAV software. Demonstrated on Boeing Unmanned Little Bird helicopter.

What Drives Price Variation?

Cost Multipliers

  • Certification level: DAL A / ASIL D / SIL 4 = highest premium
  • Tool qualification kits: Pre-packaged DO-330/ISO 26262 artifacts add $10K–$50K
  • Seat count: Enterprise floating licenses scale linearly
  • Professional services: Integration, training, certification consulting
  • Soundness guarantee: Sound analysis (Astrée) commands premium over heuristic analysis

Cost Reducers

  • Academic licensing: Many tools free or heavily discounted for universities
  • Open source alternatives: Frama-C, Infer (Meta), SonarQube Community
  • Multi-year commitments: 15–25% discounts on 3–5 year deals
  • Bundling: Polyspace cheaper when bundled with MATLAB/Simulink
  • Lower integrity levels: ASIL A/B significantly less expensive than ASIL D

What is the key difference between sound abstract interpretation (Astrée) and heuristic SAST (Coverity)?
Sound analysis proves the absence of bugs (zero false negatives) — if no error is reported, there is mathematically no error. Heuristic SAST finds bugs but cannot guarantee their absence — it may miss real bugs (false negatives) and report non-bugs (false positives).
