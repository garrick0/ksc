# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the KindScript project.

## Numbering Convention

ADRs are numbered with a three-digit prefix: `NNN-short-title.md`.

## Status Lifecycle

- **PROPOSED** — Under discussion, not yet adopted
- **ACCEPTED** — Adopted and in effect
- **SUPERSEDED** — Replaced by a later ADR (link to successor)
